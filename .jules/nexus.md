## 2026-06-20 — Entity Cross-Reference Engine
**Product understood as:** A multi-agent case-linkage and evidence-triage system for cold cases.
**Derivation reasoning:** We store case entities (people, vehicles, weapons, locations) but never explicitly highlight when the exact same entity appears across multiple isolated cases, even though this is a foundational investigative signal.
**Feature built:** An automated Entity Cross-Reference Engine that scans all cases, groups matching entities, and surfaces these overlaps in a dedicated dashboard view.
**User impact:** Investigators can instantly spot when the same suspect, vehicle, or weapon is linked to multiple seemingly unrelated cases, skipping hours of manual cross-referencing.
**Next logical feature:** Automated alert generation when a new case introduces an entity that already exists in a cold case.
